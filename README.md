## 🎯 Objetivo do Projeto

A intenção deste site é proporcionar uma experiência completa para o cliente Tenda Mar:
1. **Mensagem de Boas-vindas**: Uma introdução visualmente impactante que reforça os valores de leveza e estilo da marca.
2. **Guia de Montagem**: Um tutorial dinâmico e interativo (com vídeo integrado) para ensinar o cliente a montar sua tenda de forma simples e rápida.

---

## 📂 Estrutura do Repositório

O projeto foi desenvolvido focando em organização profissional e performance:
* **`📂 assets/`**: Armazena todos os recursos visuais, incluindo logo e imagens de alta qualidade.
* **`📄 index.html`**: Estrutura semântica e hierarquia visual da aplicação.
* **`📄 style.css`**: Design system completo com animações, responsividade e estética "luxury minimalist".
* **`📄 script.js`**: Lógica de interatividade e efeitos de reveal durante o scroll.
